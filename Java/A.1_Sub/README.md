# طرح عددين

### وصف التحدي

قم بكتابة `function` تستقبل عددين، العدد الأول يمثل رقماً صحيحاً `integer` والعدد الثاني يمثل أيضا رقماً صحيحاً `integer` ، ثم قم بإرجاع حاصل **طرح** هذين العددين.

### المخرجات المتوقعة

| Output |  b   |  a   |
| :----: | :--: | :--: |
|   5    |  5   |  10  |
|   6    |  -3  |  3   |
|   -5   |  1   |  -4  |
|   1    |  -1  |  0   |
|   0    |  0   |  0   |
|  -92   |  -4  | -96  |
